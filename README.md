# Netflix Clone

A responsive Netflix landing page clone built using HTML and CSS. The project recreates the look and feel of Netflix's homepage while focusing on responsive web design.

## Features

- Responsive layout
- Modern UI inspired by Netflix
- Navigation bar
- Multiple content sections
- Footer

## Tech Stack

- HTML
- CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/rayyanali44/netflix-clone.git
```

Open `index.html` in your browser.
